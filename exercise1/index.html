type specimen web page

overview

a project for creative computing last semester, we had to create a digital type specimen for an actual typeface of a contemporary type design foundry.

requirements

  1. typeface must contain at least 7 styles. it must be a web version of the typeface.
  2. at minimum, your web page should have:
    3. _typeface family name, type designer, + publication date
    _8 style names with example text in each style that uses the same pangram
    _a section showing all the basic uppercase + lowrecase alphabet, numerals, + basic punctuation
    _one paragraph of actual text repeated 3 times. each at different font-size + line-height settings
  3. motion + interaction. your final webpage must employ at least 2 different css animations

creative brief

i created a type specimen for lo res, designed by zuzana licko in 1985 & 2001 for the emigre, inc. design foundry. for the webpage, i kept a consistent color palette, a dark theme with a hint of neon green. to mimick the pixelated/bitmap font. i added some pixelated b&w images to make the web page interesting. i've also added a song to help set a tone (plus it was my favorite song @ the time.)
